# FAO Association Rules – FP-Growth vs Apriori

Notebook pentru preprocesarea datelor FAO GIFT și extragerea regulilor de asociere folosind FP-Growth și Apriori.

## Dataset
FAO GIFT – Romania 2012 – DIETA PILOT Adults Study.
Fișierul CSV nu este inclus în repository (dimensiune mare). Se încarcă manual în Colab.

## Rulare
Deschide `association_rules_fao.ipynb` în Google Colab și încarcă fișierul `consumption_user.csv` când apare prompt-ul.
